## [DDNSTO](https://www.ddnsto.com)

提供基于浏览器的便捷远程域名访问，远程电脑，远程文件管理等

本项目仅仅用户问题跟踪。

### 功能

* 基于浏览器的远程域名
* 基于浏览器的远程文件
* 基于浏览器的远程下载
* 基于浏览器的远程桌面

### 特性

* 配置更简单
* 更安全
* 实时响应更快

### 服务器端支持 

* iStoreOS
* Merlin
* 群晖
* QNAP
* Windows
* macOS
* Linux
